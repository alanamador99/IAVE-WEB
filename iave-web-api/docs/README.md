# 📚 Documentación del Backend - Proyecto IAVE WEB

Bienvenido a la documentación técnica del backend de IAVE. Esta documentación está organizada para facilitar el acceso a la información de controladores, base de datos y flujos del sistema.

## 🗂️ Estructura de la Documentación

La documentación se divide en las siguientes secciones principales:

### 1. 🎮 [Controladores y API](./controllers/RESUMEN_EJECUTIVO_CONTROLADORES.md)
Documentación detallada de los endpoints, lógica de negocio y controladores del sistema.

- **[Índice de Controladores](./controllers/INDICE_CONTROLADORES.md)**: Lista completa de controladores y endpoints.
- **Detalle por Módulo**:
  - [Tags](./controllers/DOCUMENTACION_TAGS.md) - Gestión de dispositivos.
  - [Casetas](./controllers/DOCUMENTACION_CASETAS.md) - Gestión de peajes y rutas.
  - [Abusos](./controllers/DOCUMENTACION_ABUSOS.md) - Infracciones y reglas.
  - [Aclaraciones](./controllers/DOCUMENTACION_ACLARACIONES.md) - Reclamos y disputas.
  - [Cruces](./controllers/DOCUMENTACION_CRUCES.md) - Registro de eventos.
  - [Sesgos](./controllers/DOCUMENTACION_SESGOS.md) - Detección de anomalías.
  - [Exportación](./controllers/DOCUMENTACION_EXPORT.md) - Generación de reportes.

### 2. 🗄️ [Base de Datos y Modelado](./database/RESUMEN_TABLAS.md)
Esquemas, diagramas y diccionarios de datos.

- **[Mapeo Completo](./database/MAPEO_TABLAS.md)**: Definición exhaustiva de tablas y atributos.
- **[Matriz de Operaciones](./database/MATRIZ_OPERACIONES.md)**: Relación entre operaciones del sistema y tablas afectadas.
- **[Diagrama y Referencia](./database/DIAGRAMA_Y_REFERENCIA.md)**: Visualización de relaciones.
- **[Índice de Tablas](./database/INDICE_TABLAS.md)**: Acceso rápido a definiciones.

### 3. 📋 [Resumen Ejecutivo](./RESUMEN_EJECUTIVO_GENERAL.md)
Visión general del proyecto, objetivos y alcance.
